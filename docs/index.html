Project Overview

The Challenge

Can we predict how players will rate a video game before it releases? User ratings strongly influence purchasing decisions and funding choices, but they're only available post-launch. We tackle this problem using two complementary machine learning approaches.

Our Approach

We built two models: a Wide & Deep MLP for flexible prediction and a Bayesian regression model for interpretability. Both achieved ~27% improvement over baseline, validating they captured real patterns in game ratings.

Methods

Wide & Deep MLP

Python / NumPy

A manually-implemented neural network inspired by Google's Wide & Deep architecture. The deep path learns smooth, non-linear patterns through hidden layers, while the wide path captures specific Platform × Genre interactions.

  • Architecture: 32 → 16 neurons (deep) + linear (wide)
  • Training: Mini-batch gradient descent, L2 regularization
  • Implementation: Manual backpropagation from scratch
  • Performance: 10.6 RMSE (28% improvement)

Bayesian Regression

R / brms

A probabilistic linear model using MCMC inference to provide interpretable coefficients with uncertainty quantification. Reveals which features most strongly predict ratings.

  • Inference: MCMC with 4 chains, 2000 iterations
  • Priors: Weakly informative (Normal, Student-t)
  • Advantage: Interpretable coefficients + uncertainty
  • Performance: 10.9 RMSE (26% improvement)

Key Findings

1

Critic Score Dominates

Critic score is by far the strongest predictor, adding ~9 points per standard deviation increase. Professional reviews strongly correlate with user satisfaction.

2

Genre & Platform Matter

Sports games are rated 4.7 points lower on average. PS2 games show a positive effect (+2.7), while PC games trend lower (-4.3).

3

Similar Model Performance

Both models achieved ~27% improvement, suggesting the relationship between features and ratings is largely linear with some valuable non-linear interactions.

4

Survivorship Bias

Older games show higher ratings (-2.3 points per year newer), likely because only highly-rated older games remain popular enough to be reviewed.

Dataset

Numeric Features

Year of Release, Critic Score, Critic Count, User Count, NA/EU/JP/Other/Global Sales

Categorical Features

Platform (PS2, Xbox, PC, etc.), Genre (Action, RPG, Sports, etc.), ESRB Rating (E, T, M, etc.)
